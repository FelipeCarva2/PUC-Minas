02.) Fazer as conversões de binário para decimal: 
02a.) manualmente  
a.) 10011  = 19 

	 1*2^(4) + 0*2^(3) + 0*2^(2) + 1*2^(1) + 1*2^(0) = 16 + 0 + 0 + 2 + 1 = 19 

b.) 10101  =  21

	1*2^(4) + 0*2^(3) + 1*2^(2) + 0*2^(1) + 1*2^(0)	= 16 + 0 + 4 + 0 + 1 = 21

c.) 10110  = 22
d.) 110010 = 50
e.) 111011 = 59

As questões c.) d.) e.) foram resolvidas seguindo o exemplo da a.) e b.)
